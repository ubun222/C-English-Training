# C-English-Training
1. 这是一个用C重写的，依赖终端的，背单词程序。支持ish，termux等移动端，也支持macOS，Linux等桌面端。
2. 不支持windows的cmd和powershell终端。若要在win下编译运行，请安装WSL。
---

### 使用方法
```
gcc c12.c
./a.out
```

### 参数
1. -i 优化iOS下的ish
2. -r 错题集模式

### tips
1. 红黄绿指示亮起时，按y/v可分别获取txt文件中的详细释义和例句，按s跳过。
